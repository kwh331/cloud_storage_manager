Написати короткий опис функціональних можливостей API

Для своєї курсової роботи я планую використовувати API від GoogleDrive, OneDrive та Dropbox.
 Функціональні можливості даних API дозволяють користувачу через відкритий протокол авторизації OAuth
 надати обмежений доступ додатку до файлів, які зберігаються на вищезгадних хмарних сховищах. Це означає
 повний контроль над даними користувача відповідно до політики додатку, зокрема, API дозволяє синхронізувати,
 видаляти, модифікувати файли, до яких був наданий доступ.
